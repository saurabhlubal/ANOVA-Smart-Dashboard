# 📊 ANOVA Smart Dashboard
 
A GUI-based data analysis tool built using Python to perform ANOVA testing, visualize data, and generate insights.
 
---
 
## 🚀 Features
- 📥 Dynamic group input
- 📊 Statistical analysis (Mean, Std Dev, Min, Max)
- 📈 Graphs (KDE Curve, Box Plot, Bar Chart)
- 📄 Final Report with insights
- 🖥️ Interactive GUI using Tkinter
 
---
 
## 🛠️ Tech Stack
- Python
- NumPy
- SciPy
- Matplotlib
- Seaborn
- Tkinter
