📊 ANOVA Smart Dashboard

An advanced, GUI-driven data analysis application designed to perform Analysis of Variance (ANOVA) with powerful visualizations and automated insights.
Built using Python, this tool simplifies complex statistical analysis into an interactive and user-friendly experience.

🚀 Key Features

✨ Dynamic Data Input

Add multiple groups seamlessly for flexible analysis
Designed for real-world datasets and experimentation

📊 Comprehensive Statistical Analysis

Mean, Standard Deviation, Minimum, Maximum
Accurate and fast computation using optimized libraries

📈 Interactive Data Visualizations

KDE Curve for distribution analysis
Box Plot for outlier detection
Bar Chart for comparative insights

📄 Automated Insightful Reports

Generates clear and concise interpretations of ANOVA results
Helps users understand statistical significance easily

🖥️ User-Friendly GUI

Built with Tkinter for a clean and interactive interface
No prior coding knowledge required to operate
🛠️ Tech Stack
Python – Core programming language
NumPy – Efficient numerical computations
SciPy – Statistical analysis (ANOVA testing)
Matplotlib – Data visualization
Seaborn – Advanced statistical plotting
Tkinter – Graphical User Interface
⚙️ Installation

Install all required dependencies using:

pip install numpy scipy matplotlib seaborn
🎯 Project Objective

The goal of this project is to bridge the gap between statistical theory and practical implementation by providing an intuitive platform where users can:

Perform ANOVA without complex coding
Visualize data distributions effectively
Interpret results with clarity
